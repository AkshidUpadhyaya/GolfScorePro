# GolfScore Pro

Java Swing desktop application (NetBeans) for managing golfers and golf rounds.

## How to Run (NetBeans)
1. Download/clone this repository.
2. Open NetBeans → File → Open Project.
3. Select the project folder.
4. Click Clean and Build.
5. Run the project.

## Features
- Golfer Management: Add / Delete / Undo Delete
- Round Management: Add / Delete
- Search & Sort: Bubble Sort + Binary Search
- Input validation with JOptionPane messages
